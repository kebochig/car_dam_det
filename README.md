# car_dam_det
 API to call car damage detector model in roboflow

Clone this repo and install the development dependencies:

```shell
# UNIX / MacOS
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

# or: Windows
py -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```
